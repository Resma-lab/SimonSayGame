🎮 Simon Says Game

A fun mini JavaScript memory challenge built to understand front-end logic and interactive programming.

🧠 Project Overview

This Simon Says Game is a browser-based interactive memory game created using HTML, CSS, and JavaScript.
It’s designed as a beginner-friendly project to help understand how JavaScript interacts with the DOM, manages user input, and simulates game logic — the foundation for learning how front-end and backend logic connect.

✨ Features

🎨 Colorful Interface: Four bright color buttons — Yellow, Red, Blue, Green — for visual appeal.

🔁 Dynamic Gameplay: Each round adds a new random color to the sequence.

🧠 Memory Training: Tests how well you can remember and repeat color sequences.

⚡ Game Feedback: Buttons flash white when triggered, black when clicked.

📈 High Score Tracking: Keeps record of your best score in the current session.

🎮 Keyboard & Mouse Start: You can start the game by either pressing a key or clicking anywhere.

🛠️ Tech Stack

HTML5 → Page structure

CSS3 → Styling and button animations

JavaScript (Vanilla) → Game logic, event listeners, DOM manipulation, score tracking

⚙️ How It Works

Start the Game:
Press any key or click anywhere to begin.

Game Sequence:
A random color flashes — you must remember the order.

User Turn:
Click the buttons in the same sequence shown.

Level Up:
If correct, you advance to the next level, and the pattern grows longer.

Game Over:
A wrong click flashes a red background and shows your score.

High Score:
If you beat your previous record, it congratulates you with a “Woo Hoo!” message.
